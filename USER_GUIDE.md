# 📚 User Guide - AnyWay to English

## 🎯 Швидкий старт

### Для викладача

#### 1️⃣ Перший вхід
1. Відкрийте сайт
2. Увійдіть як викладач
3. Система автоматично завантажить початкові дані:
   - Ціни на уроки
   - Правила платформи
   - Приклади відгуків

#### 2️⃣ Додавання студентів
1. Перейдіть в "Студенти" → "Додати студента"
2. Заповніть форму:
   - **Ім'я:** Повне ім'я студента
   - **Email:** Для входу в систему
   - **Рівень:** A0-C2
   - **Вік:** (необов'язково)
   - **Цілі:** Чому вчить англійську
   - **Наступний урок:** Дата та час
3. Натисніть "Зберегти"

#### 3️⃣ Додавання матеріалів

**Варіант A: Google Drive посилання**
1. "Бібліотека матеріалів" → "Додати матеріал"
2. Вставте посилання з Google Drive:
   ```
   https://drive.google.com/file/d/1ABC123/view
   ```
3. Система автоматично:
   - Визначить тип файлу (PDF/Video/Audio)
   - Конвертує в embeddable формат
   - Додасть preview функціональність

**Варіант B: Завантаження файлу**
1. "Бібліотека матеріалів" → "Додати матеріал"
2. Натисніть "Завантажити файл"
3. Оберіть файл (PDF, MP3, MP4, тощо)
4. Натисніть "Завантажити"
5. Файл автоматично завантажиться на Supabase Storage

**Заповніть дані:**
- **Назва:** "Present Simple - Правила"
- **Категорія:** 
  - Home Task (домашні завдання)
  - Reading (тексти для читання)
  - Grammar (граматичні вправи)
  - Vocabulary (словник)
  - Video (відео уроки)
- **Рівень:** A0-C2 або "All Levels"
- **Опис:** Короткий опис матеріалу
- **Призначити студентам:**
  - "Усім студентам" - всі побачать
  - Або оберіть конкретних студентів

#### 4️⃣ Управління матеріалами
- **Редагувати:** Натисніть ✏️ на картці матеріалу
- **Видалити:** Натисніть 🗑️ (з підтвердженням)
- **Перепризначити:** Редагуйте список студентів
- **Пошук:** Використайте поле пошуку
- **Фільтр:** Оберіть категорію (All, Home Task, Reading, тощо)

#### 5️⃣ Відстеження прогресу
1. "Журнал оцінок" → Оберіть студента
2. Перегляньте:
   - Виконані матеріали (✅)
   - Невиконані матеріали (⬜)
   - Дата виконання
   - Статистика по категоріях

### Для студента

#### 1️⃣ Вхід в систему
1. Отримайте email та пароль від викладача
2. Увійдіть на платформу
3. Побачите персональний дашборд

#### 2️⃣ Перегляд матеріалів
**"Мої матеріали"** показує тільки ті матеріали, які призначив викладач:

🎴 **Картки матеріалів:**
- 📄 **PDF** - документи, тести, вправи
- 🎥 **Video** - відео уроки з YouTube/Drive
- 🎧 **Audio** - аудіо файли для listening
- 🔗 **Link** - зовнішні посилання (Canva, Wordwall)

#### 3️⃣ Відкриття матеріалів

**PDF документи:**
1. Натисніть на картку
2. PDF відкриється в модальному вікні
3. Прокручуйте для перегляду
4. Натисніть "Відкрити в новій вкладці" для повного перегляду

**Відео (YouTube/Vimeo):**
1. Натисніть на картку
2. Відео відтвориться в модальному вікні
3. Використайте контроли для паузи/перемотки

**Audio файли:**
1. Натисніть "Відкрити"
2. Файл відкриється/завантажиться

**Зовнішні інструменти (Canva/Wordwall):**
1. Натисніть "Відкрити в новій вкладці"
2. Виконайте завдання на платформі інструменту

#### 4️⃣ Відмічання виконання
На кожній картці є чекбокс **"Виконано"**:
1. Після виконання завдання → ✅ Поставте галочку
2. Викладач побачить ваш прогрес
3. Картка буде позначена як завершена

#### 5️⃣ Словниковий тренажер
1. "Vocabulary Trainer"
2. Переглядайте слова з 3D flip анімацією:
   - **Лицьова сторона:** Англійське слово
   - **Зворотна сторона:** Переклад + приклад
3. Натисніть "Знаю" або "Повторити"
4. Відстежуйте прогрес вивчення

#### 6️⃣ Навчальний шлях
"Мій шлях" показує:
- 📊 Загальний прогрес
- 🎯 Поточний рівень (A1, B1, тощо)
- 📅 Кількість уроків
- 🏆 Досягнення

## 🔧 Налаштування

### Зміна паролю
1. Профіль → Налаштування
2. "Змінити пароль"
3. Введіть старий та новий пароль

### Налаштування сповіщень
1. Профіль → Налаштування
2. Увімкніть/вимкніть:
   - Email сповіщення про нові матеріали
   - Нагадування про уроки
   - Оновлення прогресу

## 🆘 Часті питання (FAQ)

**Q: Чому я не бачу деякі матеріали?**
A: Викладач призначає матеріали індивідуально. Якщо чогось не вистачає - напишіть викладачу.

**Q: Як завантажити PDF для офлайн використання?**
A: Натисніть "Відкрити в новій вкладці" → Правий клік → "Save as..."

**Q: Чи можу я додати власні матеріали?**
A: Наразі тільки викладач може додавати матеріали. Зв'яжіться з ним для додавання.

**Q: Що робити, якщо відео не відтворюється?**
A: 
1. Перевірте інтернет з'єднання
2. Спробуйте відкрити в новій вкладці
3. Очистіть кеш браузера

**Q: Як скинути прогрес виконання?**
A: Зніміть галочку "Виконано". Викладач також може скинути прогрес.
